# Concrete Compressive Strength Predictor

A machine learning project that predicts the compressive strength of concrete based on its mixture components (water, cement, aggregates, etc.) and curing age. This project mathematically models the non-linear physical interactions between concrete ingredients, providing a highly accurate analytical alternative to standard 28-day physical curing tests.

## Tech Stack

* **Language:** Python
* **Data Analysis & Visualization:** Numpy, Pandas, Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn (Polynomial Regression, GridSearchCV, StandardScaler)

## Model Performance

* **R-squared (R2):** 0.90
* **Root Mean Squared Error (RMSE):** 5.21

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/Raj-249/concrete-strength-predictor.git

```

2. Install the required dependencies:

```bash
pip install -r requirements.txt

```

3. Open and execute the Jupyter Notebook:

```bash
jupyter notebook concrete.ipynb

```

## Repository Structure
* **`Concrete Compressive Strength.csv`**: The original dataset containing pre-calculated engineered features.
* **`concrete_to_raw.py`**: Python script used to clean the data and strip out engineered features to prevent data leakage.
* **`concrete.csv`**: The cleaned, raw dataset generated by the script, used for model training.
* **`concrete.ipynb`**: The main Jupyter Notebook containing exploratory data analysis, preprocessing, and model optimization.
* **`requirements.txt`**: The list of Python dependencies required to run the project.
